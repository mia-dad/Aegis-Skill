# Aegis-Skill
Aegis：古希腊神话中象征“守护、秩序、权威”的盾牌. 一个可私有化的Skill技能平台


Aegis 是一个面向大模型 Agent 的通用运行时框架（Agent Runtime Framework），用于将 Skill / Tool / Workflow 等能力从具体模型中解耦出来，使任何大模型都可以在统一的 Agent 运行机制下，表现出"具备技能"的智能体能力。

## 背景与动机
在 Claude 推出 Skills 能力后，一个常见误解是：

Skill 是模型内置能力。

但在深入分析后可以发现：

Skill 不是模型权重的一部分

Skill 是通过 Agent + Runtime + 结构化约定 赋予模型的能力

Skill 本质上是一种：

可声明
可调度
可组合
可解释 的 Agent 行为模块
这意味着：

即便不使用 Claude 模型，也可以构建"具备 Claude Skill 行为表现"的 Agent。

Aegis  正是为此而生。
